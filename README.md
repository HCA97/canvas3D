Canvas teapot
=============

Utah teapot 3D HTML5 canvas realization.

Features:
* 3D moving object
* 3D rotating
* Direct illumination
* Highlighting edges


#### Requirements:
Internet Explorer 9
or
Firefox 2.0
or
Safari 3.1
or
Opera 9.0
or
Safari (Mobile) 3.2
or
Opera (Mobile) 10.0
or
Android Browser 2.1
